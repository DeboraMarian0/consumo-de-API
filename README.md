# 🇧🇷 Consulta de Feriados Nacionais

Este projeto foi desenvolvido para a disciplina de **Desenvolvimento Web II**. O objetivo principal é praticar o **consumo de APIs REST** e a manipulação dinâmica do DOM utilizando JavaScript.

A aplicação permite consultar os feriados nacionais brasileiros de qualquer ano entre 1900 e 2199, utilizando dados em tempo real.

---

## Tecnologias Utilizadas

O projeto foi construído com as seguintes tecnologias:

* **HTML5**: Estruturação semântica.
* **CSS3**: Estilização personalizada e layouts com Flexbox.
* **Bootstrap 5**: Componentes de interface e responsividade.
* **JavaScript (ES6+)**: Lógica de programação, requisições assíncronas e tratamento de dados.
* **[Brasil API](https://brasilapi.com.br/)**: API pública utilizada para obter os dados dos feriados.

---

## Funcionalidades

- [x] **Busca por Ano**: Filtro por ano específico.
- [x] **Validação**: Mensagem de erro para anos fora do intervalo suportado (1900-2199).
- [x] **Formatação de Datas**: Conversão do formato ISO (AAAA-MM-DD) para o padrão brasileiro (DD/MM).
- [x] **Responsividade**: Interface adaptável para dispositivos móveis e desktop.

---

## Aprendizados

Neste projeto, coloquei em prática conceitos fundamentais de desenvolvimento front-end:

1.  **Fetch API**: Uso do método `fetch()` para realizar chamadas HTTP GET.
2.  **Promises**: Tratamento de respostas e erros assíncronos com `.then()` e `.catch()`.
3.  **Manipulação de Arrays**: Uso do método `.map()` para percorrer os dados da API e renderizá-los como itens de lista (`<li>`) no HTML.
